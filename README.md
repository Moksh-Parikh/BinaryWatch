# What Is It
The binary watch is a digital watch powered by the attiny85
## Modes
>[!TIP]
A double click advances between modes

The first mode is minutes-only. This lights up just the inner 6 LEDs to show the minutes, in order to save power.
The second, and default, mode is full-time. This displays the hours and minutes as usual
Third is the time set mode. At first, one click will increase the minutes by 1, and a double click will enter hour set mode. Then, one click increases the hours. Another double click advances to the next mode.
Fourth is Timer/Alarm set. This mode works the same way as time set, but it instead sets an alarm.
Fifth, sixth, seventh and eighth mode support is coming soon.

>[!TIP]
If you ever lose which mode you're in, a triple click takes you back to minutes-only
